# Announce Command

{% code title="Command Syntax" %}
```
> announce HEADER MESSAGE
```
{% endcode %}

{% code title="Example Command" %}
```
> announce Event The event starts in 10 minutes
```
{% endcode %}

{% hint style="warning" %}
This announcement will see every player!
{% endhint %}
